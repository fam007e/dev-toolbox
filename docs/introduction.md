# Introduction

Welcome to the **Dev-Toolbox** documentation. Dev-Toolbox is a modular and extensible CLI toolbox for developers, built with Rust. It provides a collection of tools to streamline common development tasks, from analyzing GitHub repositories to inspecting Unicode characters.

## Key Features
- **Modular Design**: Easily extend the toolbox with new tools.
- **Cross-Platform**: Full support for Linux and Windows.
- **Global Search**: Navigate instantly with the `Ctrl+F` search palette.
- **Security First**: Integrated GitHub token scope validation and credential protection.

Explore the sidebar to learn more about the available tools and how to contribute!
